Declaring Variables:

Name the variable, use the assignment operator "=", and assign a value to the variable.

# Creating a variable called "message"
message = "Hello world!"

Declaring Functions:

  • Use the "def" keyword
  • Name the function
  • Define a list of arguments
  • Finish the function signature with ":"
  • Indent the body of the function correctly
def add_two_numbers(a, b):
    sum = a + b
    return sum
